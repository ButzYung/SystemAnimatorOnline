= MMD.js

MikuMikuDance player on WebGL.

== Demo

- http://edv.sakura.ne.jp/mmd/

== Features

- PMD (model) and VMD (motion) parsers
- Shader that's almost compatible with MMD
- Camera, light and morph animation

Yet to come

- Bone animation
- Physics

Maybe to come

- Multi-model
- Direct X accessories (.x)
- Playing audio

Likely NOT to come

- Editing motions

== Dependence

- glMatrix.js <https://github.com/toji/gl-matrix>
- DataView.js <https://github.com/davidflanagan/DataView.js> (only for compatibility browsers that hasn't implemented DataView)
